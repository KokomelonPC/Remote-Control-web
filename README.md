# Remote Control Web V1

Simple front-end prototype for the ESP32 remote control project.

## Files

- `index.html`: login and dashboard UI
- `index.html`: login, register, and dashboard UI
- `styles.css`: visual design
- `app.js`: demo interactions
- `ARCHITECTURE.md`: system recommendation

## Open It

Open `index.html` in a browser.

## Notes

- This is a UI prototype only.
- Login and add-device validation are mocked in the browser.
- Final product should move auth and validation to backend APIs.
